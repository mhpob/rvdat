# Return description fields ("template") of different VDAT data types and/or files

Return description fields ("template") of different VDAT data types
and/or files

## Usage

``` r
vdat_template(format = "csv.fathom", print = FALSE, ...)
```

## Arguments

- format:

  file format of the template. Currently only accepts "csv.fathom"; here
  for future compatibility.

- print:

  logical. Defaults to FALS.E Print the output to console?

- ...:

  arguments passed to
  [`vdat_call`](https://mhpob.github.io/rvdat/reference/vdat_call.md).
