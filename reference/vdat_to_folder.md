# Convert a VDAT file to a folder of CSVs by data type

A wrapper around `vdat_to_csv` that just changes `folder = FALSE` to
`folder = TRUE`.

## Usage

``` r
vdat_to_folder(...)
```

## Arguments

- ...:

  arguments passed to `vdat_to`.
